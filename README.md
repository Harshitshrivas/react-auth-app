# React Auth App

A modern **React authentication application** with Signup and Login functionality, built using **MongoDB**, **Express**, and **Axios**.  
Includes dashboard redirection and automatic input clearing for enhanced user experience.

---

## ğŸ”¹ Features

- **User Signup:** Users can create a new account with email and password.  
- **User Login:** Authenticate with registered credentials.  
- **Dashboard Redirect:** Successful login redirects to a protected dashboard page.   
- **Error Handling:** Alerts for invalid credentials or server errors.  
- **Responsive UI:** Built using **Tailwind CSS** for clean and responsive design.

---

## ğŸ”¹ Tech Stack

- **Frontend:** React, Tailwind CSS, Axios, React Router  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** bcrypt for password hashing

## ğŸ”¹ Installation

1. **Clone the repository**
```bash
https://github.com/Harshitshrivas/react-auth-app.git
cd react-auth-app
```
2. **Install all required packages**
```bash
npm install
```
3. **Run the server**
```bash
node index.js
```

4. **Start the React app**
```bash
npm run dev
```
5. **Open in browser**
```bash
http://localhost:5173/
```
